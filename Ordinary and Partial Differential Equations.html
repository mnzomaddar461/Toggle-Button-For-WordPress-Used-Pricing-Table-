Linear Differential Equation

A differential equation of the form:

\[\frac{d^n y}{dx^n} + P_1 \frac{d^{n-1} y}{dx^{n-1}} + P_2 \frac{d^{n-2} y}{dx^{n-2}} + \ldots + P_n y = Q \quad \ldots (i)\]

Where \(P_1, P_2, \ldots, P_n\) and \(Q\) are functions of \(x\) or constants, is called a linear differential equation of nth order.

If \(P_1, P_2, \ldots, P_n\) are all constants (not functions of \(x\)) and \(Q\) is a function of \(x\) or a constant, then the equation is called a linear differential equation with constant coefficients.

Homogeneous Linear Differential Equation

If the right-hand term \(Q\) is identically zero, then equation (i) reduces to:

\[\frac{d^n y}{dx^n} + P_1 \frac{d^{n-1} y}{dx^{n-1}} + P_2 \frac{d^{n-2} y}{dx^{n-2}} + \ldots + P_n y = 0 \quad \ldots (ii)\]

This is called a linear homogeneous differential equation.

General Solutions Based on Roots

a) Real and Distinct Roots:

If \(m_1, m_2, \ldots, m_n\) are real and distinct roots:

\[y = c_1 e^{m_1 x} + c_2 e^{m_2 x} + c_3 e^{m_3 x} + \ldots + c_n e^{m_n x}\]

b) Real and Equal Roots:

If the roots \(m_1, m_2, m_3, \ldots, m_n\) are real and equal:

\[y = (C_1 + C_2 x + C_3 x^2 + \ldots + C_n x^{n-1})e^{mx}\]

c) Complex and Distinct Roots:

If the roots are complex and distinct (of the form \(m = a \pm bi\)):

\[y = (C_1 \cos(bx) + C_2 \sin(bx))e^{ax}\]
Where \(a\) is the real part and \(b\) is the imaginary part.

d) Complex and Repeated Roots:

\[y = [(C_1 + C_2 x)\cos(bx) + (C_3 + C_4 x)\sin(bx)]e^{ax}\]

Example Problems - Homogeneous Equations

Example 1: Solve \(D^2y - 5Dy + 6y = 0\)

Solution:

Given: \(D^2y - 5Dy + 6y = 0\)

\[\Rightarrow y(D^2 - 5D + 6) = 0\]

Let \(y = e^{mx}\) be the trial solution.

Then the auxiliary equation is:

\[e^{mx}(m^2 - 5m + 6) = 0\]

\[\Rightarrow m^2 - 5m + 6 = 0\]

\[\Rightarrow m^2 - 3m - 2m + 6 = 0\]

\[\Rightarrow m(m-3) - 2(m-3) = 0\]

\[\Rightarrow (m-3)(m-2) = 0\]

\[\Rightarrow m = 2, 3\]

General solution:

\[y = c_1 e^{2x} + c_2 e^{3x} \quad \text{(Ans)}\]

Example 2: Solve \(D^4y - D^3y - 3D^2y - 11Dy - 4y = 0\)

Solution:

Given: \(y(D^4 - D^3 - 3D^2 - 11D - 4) = 0 \quad \ldots (i)\)

Let \(y = e^{mx}\) be the trial solution.

Auxiliary equation:

\[m^4 - m^3 - 3m^2 - 11m - 4 = 0\]

Factoring:

\[(m-4)(m^3 + 3m^2 + 3m + 1) = 0\]

\[\Rightarrow (m-4)(m+1)^3 = 0\]

\[\Rightarrow m = 4, -1, -1, -1\]

General solution:

\[y = c_1 e^{4x} + (c_2 + c_3 x + c_4 x^2)e^{-x} \quad \text{(Ans)}\]

Example 3: Solve \(D^2y - 2Dy + 2y = 0\) (Complex Roots)

Solution:

Given: \(D^2y - 2Dy + 2y = 0\)


\[\Rightarrow y(D^2 - 2D + 2) = 0 \quad \ldots (i)\]

Auxiliary equation: \(m^2 - 2m + 2 = 0\)

Using quadratic formula:

\[m = \frac{-(-2) \pm \sqrt{(-2)^2 - 4 \cdot 1 \cdot 2}}{2 \cdot 1}\]

\[= \frac{2 \pm \sqrt{4-8}}{2} = \frac{2 \pm \sqrt{-4}}{2}\]

\[= \frac{2 \pm 2i}{2} = 1 \pm i\]

General solution:

\[y = (c_1 \cos x + c_2 \sin x)e^x \quad \text{(Ans)}\]

Non-Homogeneous Linear Differential Equation

Consider a differential equation of the form:

\[\frac{d^n y}{dx^n} + P_1 \frac{d^{n-1} y}{dx^{n-1}} + P_2 \frac{d^{n-2} y}{dx^{n-2}} + \ldots + P_n y = Q \quad \ldots (i)\]

Where \(P_1, P_2, \ldots, P_n\) are all constants and \(Q\) is a function of \(x\) or a constant, but \(Q \neq 0\).

The general solution is:

\[y = y_c + y_p\]

Where:

  • \(y_c\) = The complementary function (solution to homogeneous equation)
  • \(y_p\) = The particular integral

Working Rules for Finding Particular Integral (\(y_p\))

Rule 1: For \(Q = x^m\) (Polynomials)

\[\frac{1}{f(D)} x^m = [1 \pm \phi(D)]^{-1} x^m\]
Expand the operator as a binomial series to operate on the polynomial.

Rule 2: For \(Q = e^{ax}\) (Exponentials)

Condition Particular Integral Formula
If \(f(a) \neq 0\) \(y_p = \frac{1}{f(a)} e^{ax}\)
If \(f(a) = 0\) but \(f'(a) \neq 0\) \(y_p = \frac{x}{f'(a)} e^{ax}\)
If \(f(a) = f'(a) = 0\) but \(f''(a) \neq 0\) \(y_p = \frac{x^2}{f''(a)} e^{ax}\)

Rule 3: For \(Q = \cos(ax)\) or \(\sin(ax)\) (Trigonometric)

Condition Particular Integral Formula
If \(f(-a^2) \neq 0\) \(\frac{1}{f(D^2)}[\cos(ax) \text{ or } \sin(ax)] = \frac{1}{f(-a^2)}[\cos(ax) \text{ or } \sin(ax)]\)
If \(f(-a^2) = 0\) but \(f'(-a^2) \neq 0\) \(y_p = \frac{x}{f'(-a^2)}[\cos(ax) \text{ or } \sin(ax)]\)

Example Problems - Non-Homogeneous Equations

Example 1: Solve \(D^2y + 4Dy + 3y = e^{-3x}\)

Solution:

Symbolic form: \(y(D^2 + 4D + 3) = e^{-3x} \quad \ldots (i)\)

Auxiliary equation:

\[m^2 + 4m + 3 = 0\]

\[\Rightarrow (m+3)(m+1) = 0\]

\[\therefore m = -1, -3\]

Complementary function:

\[y_c = c_1 e^{-x} + c_2 e^{-3x}\]

Particular integral:

From (i): \(y_p = \frac{1}{D^2 + 4D + 3} e^{-3x}\)

Since \(f(-3) = 9 - 12 + 3 = 0\), we use Case II:

\[y_p = \frac{x}{2D + 4} e^{-3x} = \frac{x}{2(-3) + 4} e^{-3x} = \frac{x}{-2} e^{-3x} = -\frac{x}{2} e^{-3x}\]

General solution:

\[y = c_1 e^{-x} + c_2 e^{-3x} - \frac{x}{2} e^{-3x} \quad \text{(Ans)}\]

Example 2: Solve \(D^2y + y = \cos 2x\)

Solution:

\[y(D^2 + 1) = \cos 2x \quad \ldots (i)\]

Auxiliary equation: \(m^2 + 1 = 0 \Rightarrow m = \pm i\)

Complementary function:

\[y_c = c_1 \cos x + c_2 \sin x\]

Particular integral:

\[y_p = \frac{\cos 2x}{D^2 + 1} = \frac{\cos 2x}{-(2)^2 + 1} = \frac{\cos 2x}{-3} = -\frac{1}{3}\cos 2x\]

General solution:

\[y = c_1 \cos x + c_2 \sin x - \frac{1}{3}\cos 2x \quad \text{(Ans)}\]

Example 3: Solve \(D^3y + 3D^2y + 3Dy + y = e^{-x}\)

Solution:

\[y(D^3 + 3D^2 + 3D + 1) = e^{-x} \quad \ldots (i)\]

Auxiliary equation:

\[m^3 + 3m^2 + 3m + 1 = 0 \Rightarrow (m+1)^3 = 0\]

\[\therefore m = -1, -1, -1\]

Complementary function:

\[y_c = (c_1 + c_2 x + c_3 x^2)e^{-x}\]

Particular integral:

Since \(f(-1) = f'(-1) = f''(-1) = 0\), use repeated differentiation:

\[y_p = \frac{x^3}{6} e^{-x}\]

General solution:

\[y = (c_1 + c_2 x + c_3 x^2)e^{-x} + \frac{x^3}{6} e^{-x} \quad \text{(Ans)}\]

Additional Examples

Example 4: Solve \(D^3y - 6D^2y + 11Dy - 6y = 0\)

Solution:

\[y(D^3 - 6D^2 + 11D - 6) = 0\]

Auxiliary equation: \(m^3 - 6m^2 + 11m - 6 = 0\)

Factoring:

\[(m-3)(m^2 - 3m + 2) = 0\]

\[(m-3)(m-2)(m-1) = 0\]

\[m = 3, 2, 1\]

General solution:

\[y = C_1 e^{3x} + C_2 e^{2x} + C_3 e^x \quad \text{(Ans)}\]

Example 5: Solve \(D^3y - 6D^2y + 11Dy - 6y = e^{2x}\)

Solution:

\[y(D^3 - 6D^2 + 11D - 6) = e^{2x} \quad \ldots (i)\]

From previous example: \(m = 3, 2, 1\)

Complementary function:

\[y_c = C_1 e^{3x} + C_2 e^{2x} + C_3 e^x\]

Particular integral:

\[y_p = \frac{1}{D^3 - 6D^2 + 11D - 6} e^{2x}\]

Since \(f(2) = 8 - 24 + 22 - 6 = 0\), use Case II:

\[y_p = \frac{x}{3D^2 - 12D + 11} e^{2x} = \frac{x}{3(4) - 12(2) + 11} e^{2x}\]

\[= \frac{x}{12 - 24 + 11} e^{2x} = \frac{x}{-1} e^{2x} = -x e^{2x}\]

General solution:

\[y = C_1 e^{3x} + C_2 e^{2x} + C_3 e^x - x e^{2x} \quad \text{(Ans)}\]

Example 6: Solve \(D^4y - D^3y - 9D^2y - 11Dy - 4y = e^{-x}\)

Solution:

\[y(D^4 - D^3 - 9D^2 - 11D - 4) = e^{-x} \quad \ldots (i)\]

Auxiliary equation: \(m^4 - m^3 - 9m^2 - 11m - 4 = 0\)

Factoring: \((m-4)(m+1)^3 = 0\)

\[m = 4, -1, -1, -1\]

Complementary function:

\[y_c = C_1 e^{4x} + (C_2 + C_3 x + C_4 x^2)e^{-x}\]

Particular integral:

Since \(f(-1) = f'(-1) = f''(-1) = 0\), use triple differentiation:

\[y_p = \frac{x^3}{24D - 6} e^{-x} = \frac{x^3}{24(-1) - 6} e^{-x} = \frac{x^3}{-30} e^{-x} = -\frac{x^3}{30} e^{-x}\]

General solution:

\[y = C_1 e^{4x} + (C_2 + C_3 x + C_4 x^2)e^{-x} - \frac{x^3}{30} e^{-x} \quad \text{(Ans)}\]

Cauchy-Euler Differential Equation

A differential equation of the form:

\[x^n \frac{d^n y}{dx^n} + P_1 x^{n-1} \frac{d^{n-1} y}{dx^{n-1}} + P_2 x^{n-2} \frac{d^{n-2} y}{dx^{n-2}} + \ldots + P_n y = Q\]

where \(P_1, P_2, \ldots, P_n\) and \(Q\) are functions of \(x\) or constants, is called a Cauchy-Euler differential equation.

Solution Method

Substitution: Let \(x = e^z\), so \(z = \ln x\)

Let \(D_1 = \frac{d}{dz}\)

Then:

  • \(xD = D_1\)
  • \(x^2D^2 = D_1(D_1 - 1)\)
  • \(x^3D^3 = D_1(D_1 - 1)(D_1 - 2)\)

Example 1: Solve \(x^3D^3y + 3x^2Dy + 25y = 0\)

Solution:


\[y(x^3D^3 + 3x^2D + 25) = 0 \quad \ldots (i)\]

Let \(x = e^z\), so \(z = \ln x\) and \(D_1 = \frac{d}{dz}\)

Then (i) becomes:

\[D_1(D_1-1) + 3D_1 + 25 = 0\]

\[(D_1^2 + 2D_1 + 25)y = 0\]

Auxiliary equation: \(m^2 + 8m + 25 = 0\)

\[m = \frac{-8 \pm \sqrt{64 - 100}}{2} = \frac{-8 \pm \sqrt{-36}}{2} = \frac{-8 \pm 6i}{2} = -4 \pm 3i\]

General solution in z:

\[y = (c_1 \cos 3z + c_2 \sin 3z)e^{-4z}\]

Substituting \(z = \ln x\):

\[y = (c_1 \cos(3\ln x) + c_2 \sin(3\ln x))x^{-4} \quad \text{(Ans)}\]

Example 2: Solve \(x^3D^3y - x^2D^2y + 2xDy - 2y = x^3\)

Solution:

\[y(x^3D^3 - x^2D^2 + 2xD - 2) = x^3 \quad \ldots (i)\]

Substituting \(x = e^z\):

\[(D_1^3 - 4D_1^2 + 5D_1 - 2)y = e^{3z}\]

Auxiliary equation: \(m^3 - 4m^2 + 5m - 2 = 0\)

Factoring: \((m-2)(m-1)^2 = 0\)

\[m = 2, 1, 1\]

Complementary function:

\[y_c = (c_1 + c_2 z)e^z + c_3 e^{2z}\]

\[= (c_1 + c_2 \ln x)x + c_3 x^2\]

Particular integral:

\[y_p = \frac{1}{27 - 36 + 15 - 2} e^{3z} = \frac{1}{4} e^{3z} = \frac{x^3}{4}\]

General solution:

\[y = (c_1 + c_2 \ln x)x + c_3 x^2 + \frac{x^3}{4} \quad \text{(Ans)}\]

Example 3: Solve \(x^2D^2y - 2xDy - 4y = 0\)

Solution:

\[y(x^2D^2 - 2xD - 4) = 0 \quad \ldots (i)\]

After substitution: \(D_1^2 - 3D_1 - 4 = 0\)

\[(D_1 - 4)(D_1 + 1) = 0\]

\[m = 4, -1\]

General solution:

\[y = c_1 x^4 + c_2 x^{-1} \quad \text{(Ans)}\]

Partial Differential Equations

Notation

For \(z = f(x, y)\):

  • \(p = \frac{\partial z}{\partial x}\) (first partial derivative with respect to \(x\))
  • \(q = \frac{\partial z}{\partial y}\) (first partial derivative with respect to \(y\))
  • \(r = \frac{\partial^2 z}{\partial x^2} = f_{xx}\) (second partial derivative w.r.t. \(x\))
  • \(s = \frac{\partial^2 z}{\partial x \partial y} = f_{xy}\) (mixed partial derivative)
  • \(t = \frac{\partial^2 z}{\partial y^2} = f_{yy}\) (second partial derivative w.r.t. \(y\))

Formation of Partial Differential Equations

PDEs are formed by eliminating arbitrary constants or functions from a given relation between variables.

Case (i): Elimination of Arbitrary Constants

Example 1: Form a PDE from \(z = (x-a)^2 + (y-b)^2\)

Solution:

Given: \(z = (x-a)^2 + (y-b)^2\)

Differentiating partially:

\[\frac{\partial z}{\partial x} = 2(x-a)\]

\[\frac{\partial z}{\partial y} = 2(y-b)\]

Squaring and adding:

\[\left(\frac{\partial z}{\partial x}\right)^2 + \left(\frac{\partial z}{\partial y}\right)^2 = 4[(x-a)^2 + (y-b)^2] = 4z\]

PDE:

\[\left(\frac{\partial z}{\partial x}\right)^2 + \left(\frac{\partial z}{\partial y}\right)^2 = 4z \quad \text{(Ans)}\]

Example 2: Form a PDE from \(z = a(x+y) + b(x-y) + ab + c\)

Solution:

Given: \(z = a(x+y) + b(x-y) + ab + c\)

Differentiating:

\[\frac{\partial z}{\partial x} = a + b\]

\[\frac{\partial z}{\partial y} = a - b\]

\[\frac{\partial z}{\partial t} = ab\]

Squaring the first two and subtracting:


\[\left(\frac{\partial z}{\partial x}\right)^2 - \left(\frac{\partial z}{\partial y}\right)^2 = (a+b)^2 - (a-b)^2 = 4ab\]

PDE:

\[\left(\frac{\partial z}{\partial x}\right)^2 - \left(\frac{\partial z}{\partial y}\right)^2 = 4\frac{\partial z}{\partial t} \quad \text{(Ans)}\]

Example 3: Sphere of radius 4 with center on xy-plane

Solution:

Given: \((x-a)^2 + (y-b)^2 + z^2 = 16\)

Differentiating partially:

\[2(x-a) + 2z\frac{\partial z}{\partial x} = 0 \Rightarrow (x-a) = -z\frac{\partial z}{\partial x}\]

\[2(y-b) + 2z\frac{\partial z}{\partial y} = 0 \Rightarrow (y-b) = -z\frac{\partial z}{\partial y}\]

Substituting back:

\[\left(-z\frac{\partial z}{\partial x}\right)^2 + \left(-z\frac{\partial z}{\partial y}\right)^2 + z^2 = 16\]

PDE:

\[z^2\left[\left(\frac{\partial z}{\partial x}\right)^2 + \left(\frac{\partial z}{\partial y}\right)^2\right] + z^2 = 16 \quad \text{(Ans)}\]

Example 4: Planes with intercepts summing to zero

Solution:

Given: \(\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1\) and \(a + b + c = 0\)

Let \(p = \frac{dz}{dx}\) and \(q = \frac{dz}{dy}\)

From differentiation:

\[\frac{1}{a} + \frac{p}{c} = 0 \Rightarrow c = -ap\]

\[\frac{1}{b} + \frac{q}{c} = 0 \Rightarrow b = -\frac{c}{q}\]

Eliminating \(a\), \(b\), \(c\) using determinant method:

PDE:

\[\frac{dz}{dx} + \frac{dz}{dy} = \frac{dz}{dx} \cdot \frac{dz}{dy} \quad \text{(Ans)}\]

Case (ii): Elimination of Arbitrary Functions

Example 1: \(z = xy + f(x^2 + y^2)\)

Solution:

Given: \(z = xy + f(x^2 + y^2)\)

Differentiating:

\[\frac{\partial z}{\partial x} = y + 2xf'(x^2 + y^2)\]

\[\frac{\partial z}{\partial y} = x + 2yf'(x^2 + y^2)\]

Eliminating \(f'\):

\[y\frac{\partial z}{\partial x} - x\frac{\partial z}{\partial y} = y^2 - x^2\]

PDE:

\[y\frac{\partial z}{\partial x} - x\frac{\partial z}{\partial y} = y^2 - x^2 \quad \text{(Ans)}\]

Example 2: \(z = x + y + f(xy)\)

Solution:

Given: \(z = x + y + f(xy)\)

Differentiating:

\[\frac{\partial z}{\partial x} = 1 + yf'(xy)\]

\[\frac{\partial z}{\partial y} = 1 + xf'(xy)\]

Eliminating \(f'\):

PDE:

\[x\frac{\partial z}{\partial x} - y\frac{\partial z}{\partial y} = x - y \quad \text{(Ans)}\]

Example 3: \(x + y + z = f(x^2 + y^2 + z^2)\)

Solution:

Given: \(x + y + z = f(x^2 + y^2 + z^2)\)

Let \(p = \frac{\partial z}{\partial x}\) and \(q = \frac{\partial z}{\partial y}\)

Differentiating:

\[1 + p = f'(x^2 + y^2 + z^2) \cdot (2x + 2zp)\]

\[1 + q = f'(x^2 + y^2 + z^2) \cdot (2y + 2zq)\]

Eliminating \(f'\):

PDE:

\[(1 + p)(2y + 2zq) = (1 + q)(2x + 2zp) \quad \text{(Ans)}\]

Example 4: \(xyz = f(x + y + z)\)

Solution:

Given: \(xyz = f(x + y + z)\)

Differentiating and eliminating \(f'\):

PDE:

\[xy\left(x\frac{dz}{dx} + z\right)\left(1 + \frac{dz}{dy}\right) = x\left(y\frac{dz}{dy} + z\right)\left(1 + \frac{dz}{dx}\right) \quad \text{(Ans)}\]

Example 5: \(z = e^{ax + iby} \cdot f(ax - by)\)

Solution:

Given: \(z = e^{ax + iby} \cdot f(ax - by)\)

Differentiating and eliminating \(f'\):

PDE:

\[b\frac{\partial z}{\partial x} + a\frac{\partial z}{\partial y} = 2abz \quad \text{(Ans)}\]

Example 6: \(z = f(x^2 - y) + g(x^2 + y)\)

Solution:

Given: \(z = f(x^2 - y) + g(x^2 + y)\)

Differentiating twice and eliminating \(f''\) and \(g''\):

PDE:

\[x\frac{\partial^2 z}{\partial x^2} - 4x^3\frac{\partial^2 z}{\partial y^2} = \frac{\partial z}{\partial x} \quad \text{(Ans)}\]

Example 7: \(z = f(x + iy) + g(x - iy)\)

Solution:

Given: \(z = f(x + iy) + g(x - iy)\)

Differentiating twice:

\[\frac{\partial^2 z}{\partial x^2} = f''(x + iy) + g''(x - iy)\]

\[\frac{\partial^2 z}{\partial y^2} = -f''(x + iy) - g''(x - iy)\]

PDE (Laplace's equation in complex form):

\[\frac{\partial^2 z}{\partial y^2} = -\frac{\partial^2 z}{\partial x^2} \quad \text{(Ans)}\]

Classification of Second-Order PDEs

For a general second-order PDE:

\[AU_{xx} + BU_{xy} + CU_{yy} + FU_x + GU_y + HU = E\]

The discriminant is: \(\Delta = B^2 - 4AC\)

Classification:

  • Hyperbolic: \(\Delta > 0\) (e.g., Wave equation)
  • Elliptic: \(\Delta < 0\) (e.g., Laplace's equation)
  • Parabolic: \(\Delta = 0\) (e.g., Heat equation)

Example 1: Classify \(U_{xx} + U_{yy} + 4U_x = 0\)

Solution:

\(A = 1\), \(B = 0\), \(C = -1\)

\[\Delta = 0^2 - 4(1)(-1) = 4 > 0\]

Classification: Hyperbolic

Example 2: Classify \(2U_{xx} + 3U_{xy} + 4U_{yy} = 0\)

Solution:

\(A = 2\), \(B = 3\), \(C = 4\)

\[\Delta = 3^2 - 4(2)(4) = 9 - 32 = -23 < 0\]

Classification: Elliptic

Example 3: Classify \(yU_{xx} - xU_{xy} + yU_{yy} - 8x^2U_y = xe^{-2y}\)

Solution:

\(A = y\), \(B = -x\), \(C = y\)

\[\Delta = (-x)^2 - 4(y)(y) = x^2 - 4y^2 = (x+2y)(x-2y)\]

Classification depends on region:

  • Hyperbolic: \(x < -2y\) or \(x > 2y\)
  • Elliptic: \(-2y < x < 2y\)
  • Parabolic: \(x = \pm 2y\)
